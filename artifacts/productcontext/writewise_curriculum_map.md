# WriteWise Kids — Complete Curriculum Map

## Overview

This curriculum covers four core writing types across three developmental tiers, providing a comprehensive K-8 writing education framework.

**Writing Types:**
- Narrative Writing
- Persuasive/Opinion Writing
- Expository/Informational Writing
- Descriptive Writing

**Developmental Tiers:**
- Tier 1: Ages 7-9 (Grades 2-4)
- Tier 2: Ages 10-12 (Grades 5-7)
- Tier 3: Ages 13-15 (Grades 8-10)

---

## Tier 1: Foundational (Ages 7-9)

### Learning Characteristics
- Short attention spans (10-15 minute focused activities)
- Concrete thinking; needs tangible examples
- Benefits from visual supports and graphic organizers
- Motivated by creativity, imagination, and praise
- Writing stamina: 50-150 words per session

---

### NARRATIVE WRITING — Tier 1

#### Unit N1.1: Story Beginnings
**Learning Objectives:**
- Understand that stories have a beginning that introduces characters and setting
- Learn what a "hook" is and why it matters

**Lessons:**

|--------|-------|------------|

## Skill Progression Summary Matrix

| Skill Domain | Tier 1 (7-9) | Tier 2 (10-12) | Tier 3 (13-15) |
|--------------|--------------|----------------|----------------|
| **Structure** | B-M-E, simple sequencing | Story mountain, 5-paragraph essay | Non-linear, complex structures |
| **Voice** | First vs. third person | POV consistency, basic voice | Distinctive authorial voice |
| **Detail** | Sensory words, basic description | Show don't tell, figurative language | Imagery, symbolism, atmosphere |
| **Character** | Basic traits, simple emotions | Dialogue, motivation | Psychology, arcs, internal monologue |
| **Argument** | Opinion + because | Thesis, evidence, counterargument | Nuanced argument, rhetorical analysis |
| **Research** | Facts about topics | Basic research, paraphrasing | Extended research, synthesis |
| **Grammar** | Complete sentences | Varied sentences, punctuation | Syntax for effect |
| **Revision** | Add details | Structural revision | Multi-draft process |

## Assessment Philosophy

Each lesson culminates in an assessment that:
1. Aligns directly with the lesson's learning objectives
2. Is appropriate for the tier's developmental level
3. Provides opportunity for meaningful feedback
4. Can be graded using the accompanying rubrics

Assessment types include:
- **Quick checks**: Short exercises within lessons
- **Lesson assessments**: End-of-lesson writing tasks
- **Unit projects**: Cumulative writing pieces
- **Portfolio pieces**: Best work for showcase

## Implementation Notes

### Differentiation
- Within each tier, lessons can be adjusted for struggling or advanced students
- Scaffolding can be increased or decreased based on student need
- Extension activities available for students who finish early

### Pacing
- Tier 1: ~2-3 sessions per lesson
- Tier 2: ~2-3 sessions per lesson  
- Tier 3: ~3-4 sessions per lesson

### Prerequisites
- Students should be placed based on skill, not just age
- Diagnostic assessment recommended before placement
- Movement between tiers should be supported when appropriate
