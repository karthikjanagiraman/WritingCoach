# WriteWise Kids Curriculum Map

## Overview

Four writing types across three developmental tiers.

| Tier | Ages | Grades | Writing Stamina |
|------|------|--------|-----------------|
| 1 | 7-9 | 2-4 | 50-150 words |
| 2 | 10-12 | 5-7 | 150-400 words |
| 3 | 13-15 | 8-10 | 400-800 words |

---

## Lesson ID Format

```
{Type}{Tier}.{Unit}.{Lesson}

Examples:
N1.1.4 = Narrative, Tier 1, Unit 1, Lesson 4
P2.3.2 = Persuasive, Tier 2, Unit 3, Lesson 2
```

| Prefix | Writing Type |
|--------|--------------|
| N | Narrative |
| P | Persuasive/Opinion |
| E | Expository/Informational |
| D | Descriptive |
